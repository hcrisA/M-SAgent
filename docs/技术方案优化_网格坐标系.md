# 图像分割Agent技术方案优化 - 网格坐标系

## 一、核心问题

**问题描述**：MLLM基于网格坐标系返回点坐标时，与目标差距较大。

**根因分析**：
1. MLLM的Vision Encoder在patch embedding过程中丢失细粒度空间信息
2. 自回归生成坐标存在量化误差和不连续预测问题
3. MLLM难以精确输出像素级坐标，---

## 二、优化核心思路

**保留网格坐标系**，- MLLM输出网格级BBox（粗粒度区域），- 在BBox内多点采样
- MLLM判断每个点是否在目标上（Yes/No判断）
- 筛选正负点，- SAM分割

## 七、关键代码片段

### 7.1 网格BBox转像素坐标

```python
def grid_bbox_to_pixel(self, grid_bbox, image_size):
    width, height = image_size
    row_positions = np.linspace(0, height, self.grid_rows + 1, dtype=int)
    col_positions = np.linspace(0, width, self.grid_cols + 1, dtype=int)
    
    col1, row1, col2, row2 = grid_bbox
    
    x1 = col_positions[max(0, min(col1, self.grid_cols))]
    y1 = row_positions[max(0, min(row1, self.grid_rows))]
    x2 = col_positions[max(0, min(col2, self.grid_cols))]
    y2 = row_positions[max(0, min(row2, self.grid_rows))]
    
    return [int(x1), int(y1), int(x2), int(y2)]
```

### 7.2 均匀采样点

```python
def sample_points_in_bbox(self, pixel_bbox, n=20):
    x1, y1, x2, y2 = pixel_bbox
    
    # 根据bbox长宽比计算行列数
    aspect_ratio = (x2 - x1) / (y2 - y1 + 1)
    cols = int(np.sqrt(n * aspect_ratio))
    rows = int(n / cols) if cols > 0 else 1
    
    # 均匀采样（排除边界）
    x_coords = np.linspace(x1, x2, cols + 2)[1:-1]
    y_coords = np.linspace(y1, y2, rows + 2)[1:-1]
    
    points = [[x, y] for x in x_coords for y in y_coords]
    return points[:n]
```

### 7.3 MLLM点验证

```python
def verify_points_with_mllm(self, marked_image, sample_points, text_prompt):
    positive_points = []
    negative_points = []
    
    # 批量验证（每批5个点）
    for i in range(0, len(sample_points), self.batch_size):
        batch_points = sample_points[i:i+self.batch_size]
        batch_indices = list(range(i+1, i+1+len(batch_points)))
        
        question = f"判断编号{batch_indices}的点是否在"{text_prompt}"上"
        answer = self.mllm.verify_points(marked_image, question)
        
        # 解析答案
        for j, idx in enumerate(batch_indices):
            if answer.get(str(idx), "No") == "Yes":
                positive_points.append(batch_points[j])
            else:
                negative_points.append(batch_points[j])
    
    return positive_points, negative_points
```
